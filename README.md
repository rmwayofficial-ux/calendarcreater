# 予約状況カレンダー作成アプリ

セラピストさん向け。**午前・午後のマスをタップして ◎○△× を切り替えるだけ**で、
月間の予約状況カレンダーを作り、**JPEG画像でダウンロード**できます（ブログ貼り付け用・横長）。

- サーバー不要。すべてブラウザ内で完結し、入力内容は端末のローカルにのみ保存されます。
- **日曜始まり**のカレンダー。祝日（振替休日・国民の休日含む / 1980〜2099年）を自動表示。
- カレンダー周りの余白に、**その月の季節に合った背景イラスト**（雪/梅/桜/春/新緑/梅雨/七夕/夏/月見/紅葉/銀杏/冬）を自動で添えます。

## マークの色と意味

記号は色付き・太字で表示されます（背景の塗りなし）。

| マーク | 色 | 意味 |
|:---:|:---:|---|
| ◎ | 赤 | 空き十分 |
| ○ | 緑 | 空きあり |
| △ | 青 | 残りわずか |
| × | 濃グレー | 満席 |
| 休 | グレー（細字） | 休み・定休 |

カレンダー本体の配色は、白（メイン）／teal `#4ab4aa`（アクセント：見出し・曜日帯）／グレー `#808080`（枠・罫線）。

## 使い方

```bash
npm install      # 初回のみ
npm run dev      # 開発サーバー起動（http://localhost:5173）
npm run build    # 本番ビルド（dist/ に出力 → 静的ホスティングにアップ可）
```

### 操作
1. 年月を選ぶ（◀▶ または プルダウン）
2. タイトルを入力（既定は「○月の予約状況」。見出しは大きく表示されます）
3. 各日の「AM」「PM」のマスをクリックすると `空白 → ◎ → ○ → △ → ×` と切り替わる
4. 右側のプレビューを確認して **「📥 JPEGをダウンロード」**
5. ダウンロードした画像をブログにアップ

### ことばで入力（特別な日）
各日のマスの下に**自由入力欄**があります。特別なイベントや出張など、◎○△× では表せない日は、ここにことばを入力します。
- **ことばだけ入力（マークは空白）** → その日はことばが大きく中央に表示されます（例：「大阪へ出張のため終日お休み」）
- **マーク＋ことば** → ◎○△× の下に小さくことばが表示されます（例：AM○・PM× ＋「夕方ヨガイベント」）
- 長い文は自動で折り返し、入りきらない分は「…」で省略されます

### 便利機能
- **前月をコピー**：前の月の入力内容を引き継ぎ、変更点だけ直す
- **曜日まとめ設定**：「毎週月曜は空白（定休）」などをワンタッチ
- **全部クリア**：その月のマークを一括消去
- **自動保存**：入力は自動でブラウザに保存され、次回開くと続きから

## 技術構成
- React 18 + TypeScript + Vite
- 画像生成：HTML Canvas に直接描画 → `toDataURL('image/jpeg')`
- 保存：localStorage（月ごと）
